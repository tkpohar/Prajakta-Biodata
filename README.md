# Prajakta-Biodata

This repository contains a marriage-focused biodata template (HTML + CSS) that you can personalize and publish.

Quick steps to personalize:

1. Open `index.html` and replace placeholder text (name, DOB, contact, family details) with your own details.
2. Preview locally with a simple server:

```bash
cd Prajakta-Biodata
python3 -m http.server 8000
# then open http://127.0.0.1:8000 in your browser
```

3. To get a printable copy, use your browser's Print -> Save as PDF (the template is optimized for A4).

4. To publish on GitHub Pages:

```bash
git add .
git commit -m "Add marriage biodata template"
git push
# Then enable GitHub Pages from repository settings (branch: main, folder: /)
```

If you'd like, send me your biodata details and I will fill them in and prepare a print-ready PDF for you.
